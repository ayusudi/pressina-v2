## Getting Started
**Server**
```
cd server 
npm install
touch .env
create .env with .env.template
npm run dev
baseurl is localhost:3000
```
**Client**
```
cd client-v2
npm install
npm run serve
```

for your information, REST API documentation is inside server folder.
And for client that's open in localhost doesn't have google signin.
